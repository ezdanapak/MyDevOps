    Firewall¶

    Firewall-ში HTTP-ის დაშვება UFW ჩართულია:

    sudo ufw allow 'Nginx HTTP'
    სტატუსის შემოწმება 
    sudo ufw status

    ლოკალური ტესტი სერვერზე:¶

    curl http://localhost
    თუ ამ html ფაილს დაინახავ ესეიგი ყველაფერი კარგად არის და სერვერი მუშაობს!

    IP მისამართზე ვებ სერვერი¶

    მისამართი Ubuntu server მა რაც აიღო სტანდარტულად 80 პორტზე მუშაობს ვებ სერვერი
    თუ გვინდა რომ სხვა კომპიუტერიდან(იმავე ქსელის შიდა როუტერით) მივწვდეთ სერვერს უნდა
    გამოვიყენოთ ნებისმიერ ინტერნეტ ბრაუზერში მისი IP.

    http://192.168.56.101/
