# Multifunction Project

სალამი! ეს არის **Multifunction Project** – მრავალფუნქციური პროექტის საქაღალდე, რომელიც შეიცავს სხვადასხვა რესურსებს.

## საქაღალდეების სტრუქტურა

პროექტი არის შემდეგი საქაღალდეებისგან:

### 1. `github_repo_inside`
- შიგნით არის პროექტის სრული **GitHub რეპოზიტორია**.
- გამოიყენება როგორც ძირითადი კოდი, მაგალითები და ლიცენზიები.

### 2. `Screenshot`
- საპრეზენტაციო სქრინშოტები.
- იდეალურია ვიზუალური დემოსთვის.

### 3. `UVM`
- შიგნით მოიძებნება:
  - ვირტუალური მანქანები (Virtual Machines)
  - `Mobaxterm.exe`
  - `Ubuntu Server ISO`
  - `VirtualBox.exe`
- გამოიყენება განვითარების ან ტესტირების გარემოს აღსადგენად.

### 4. `PPK`
- შეიცავს **გასაღებებს (Private/Public keys)**.
- გამოიყენება უსაფრთხო წვდომისთვის, მაგალითად VM-ებში ან რეპოზიტორიებზე.

## გამოყენება

1. გახსენით `github_repo_inside` საქაღალდე და გაეცანით README-ს იქ.
2. გადაათვალიერეთ `Screenshot` საქაღალდე საპრეზენტაციოდ.
3. VM-ების გაშვება შესაძლებელია `UVM` საქაღალდან შესაბამის ინსტრუქციას მიყვებით.
4. PPK გასაღებები გამოიყენეთ შესაბამისი შესვლისთვის.
