# 🖥️ Virtual Machine Setup – Ubuntu Server

Ubuntu Server-ის ინსტალაციისთვის გამოვიყენე **VirtualBox** Windows გარემოში.  
დადგა უახლესი ვერსია და დავაინსტალირე VM-სთვის შემდეგი პარამეტრები:

- **RAM:** 8 GB  
- **CPU:** 4 Cores  
- **Disk:** 50 GB

---

## 🌐 Network Configuration

VM-ზე შესრულებული ქსელის კონფიგურაცია:

| Adapter | ტიპი | გამოყენება |
|---------|------|------------|
| Adapter 1 | NAT | ინტერნეტ წვდომა VM-ში |
| Adapter 2 | Host-Only | SSH კავშირი Host Machine-დან |
| Adapter 3 | Bridged | ქსელის სხვა მოწყობილობებთან კომუნიკაცია (ფაკულტატური) |

> ✅ შედეგი: VM-ს აქვს ინტერნეტი, შესაძლებელია SSH-ის გამოყენება host-მეთოდიდან, ხოლო საჭიროების შემთხვევაში VM კომუნიკაციას ახდენს ქსელზე.

---

## 🔧 დამატებითი შენიშვნები

- Host-Only ქსელი აუცილებელია **დისტანციური ადმინისტრაციისთვის**.  
- Bridged ქსელი გამოყენებულია მხოლოდ როდესაც VM უნდა იყოს LAN-ზე გამოჩენილი.  
- NAT უზრუნველყოფს სწრაფ და მარტივ ინტერნეტ წვდომას VM-ში.
