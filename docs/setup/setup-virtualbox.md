# ­ЪќЦ№ИЈ Virtual Machine Setup РђЊ Ubuntu Server

Ubuntu Server-рЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћ **VirtualBox** рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃБрЃџ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃќрЃћ.  
рЃЊрЃљрЃљрЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃћ VM-рЃАрЃЌрЃЋрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў:

- **RAM:** 8 GB  
- **CPU:** 4 Cores  
- **Disk:** 50 GB

---

## ­Ъїљ Network Configuration

VM-рЃќрЃћ рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃБрЃџрЃў рЃЦрЃАрЃћрЃџрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ:

| Adapter | рЃбрЃўрЃърЃў | рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ |
|---------|------|------------|
| Adapter 1 | NAT | рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃб рЃгрЃЋрЃЊрЃЮрЃЏрЃљ VM-рЃерЃў |
| Adapter 2 | Host-Only | SSH рЃЎрЃљрЃЋрЃерЃўрЃарЃў Host Machine-рЃЊрЃљрЃю |


> РюЁ рЃерЃћрЃЊрЃћрЃњрЃў: VM-рЃА рЃљрЃЦрЃЋрЃА рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃбрЃў, рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ SSH-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ host-рЃЏрЃћрЃЌрЃЮрЃЊрЃўрЃЊрЃљрЃю, рЃ«рЃЮрЃџрЃЮ рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў VM рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃљрЃА рЃљрЃ«рЃЊрЃћрЃюрЃА рЃЦрЃАрЃћрЃџрЃќрЃћ.

---

## ­ЪћД рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃћрЃЉрЃў

- Host-Only рЃЦрЃАрЃћрЃџрЃў рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ **рЃЊрЃўрЃАрЃбрЃљрЃюрЃфрЃўрЃБрЃарЃў рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА**.    
- NAT рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃАрЃгрЃарЃљрЃц рЃЊрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋ рЃўрЃюрЃбрЃћрЃарЃюрЃћрЃб рЃгрЃЋрЃЊрЃЮрЃЏрЃљрЃА VM-рЃерЃў.
